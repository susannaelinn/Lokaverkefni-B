This project is a school equipment management system made in Rust. The program can manage desks, chairs, and projectors using an SQLite database. It supports CRUD operations, filtering, sorting, and saving data to JSON files.

The project was built using Rust, rusqlite, and serde_json. Different parts of the program were split into separate files to keep the code organized and easier to work with. Structs and enums were used to represent the different equipment types and locations.

The system can create, read, update, and delete equipment from the database. It can also filter equipment by building and sort it by building, floor, and room number. JSON functionality was added so equipment data can be saved and loaded from files.

AI tools like ChatGPT were used during development to help with Rust syntax, SQLite integration, error handling, and general project structure. Even with AI help, the code still had to be tested, debugged, and connected together manually.

Overall, the project helped improve my understanding of Rust, databases, modular programming, and error handling.
